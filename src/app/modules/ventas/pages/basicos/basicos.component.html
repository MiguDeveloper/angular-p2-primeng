Date | Fecha | Pipe

Pipes para mostrar las fechas.

  • fecha:
    • {{ fecha }}
  • fecha | date
    • {{ fecha | date }}
  • fecha | date:'short'
    • {{ fecha | date : "short" }}
  • fecha | date:'long'
    • {{ fecha | date : "long" }}
  • fecha | date:'MMMM dd, yyyy'
    • {{ fecha | date : "MMMM dd, yyyy" }}
  • fecha | date:'long':'GMT-6':'fr'
    • {{ fecha | date : "long" : "GMT-6" : "fr" }}
  • fecha | date:'long':'GMT-4'
    • {{ fecha | date : "long" : "GMT-4" }}
